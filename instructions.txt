Config settings:
	git config --list

Initalize repo:
	git init
Cloning a repo:
	git clone https://github.com/rsDevelopment/learngit.git
		-- by default it will create folder with project name. It can be overriden by passing name as last arg

	git remote -v -- to see a list of all remote repos associate with project
To enable auto-completion:
	curl https://raw.githubusercontent.com/git/git/master/contrib/completion/git-completion.zsh -o ~/.zsh/_git/git-completion.zsh

then add the following to your ~/.zshrc file:
	fpath=(~/.zsh $fpath)

Commands:
	To add a file:
		git add . [ tracks the current and any child directories ]
		or add by file name
		git add whateverfile.txt
		git add -u

	To see the contents of a hash:
		git show hashchars
		

Testing

Add remote repository:
	git remote add origin https://github.com/rsDevelopment/learngit.git

Commit to remote repository
	git push -u origin master
		The -u establishes a permanent link so that it can keep comparing when doing status

Branches
	git branch
		To see current branches
	git branch -a
		To see remote branches

	Branch creation:
		git branch branchName	
		git checkout branchName -- To switch to the branch
		git branch -r newBranchName
		git branch -d targetBranch | git branch -D targetBranch for hard delete
		git merge targetBranch

	Branch pushing
		git push origin sourceBranch:targetBranch

Pulling data:
	git fetch origin
